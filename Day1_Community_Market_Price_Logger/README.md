# Community Market Price Logger (Python)

## Problem
Food price tracking is manual and unreliable in many Nigerian communities.

## Solution
A simple Python program to log and display daily market prices.

## Features
- Store market info
- Record food prices
- Compare prices manually

## Technologies
- Python 3
- CLI (Terminal-based)

## How to Run
1. Install Python
2. Run `python market_price_logger.py`

## Future Improvements
- Save data to file
- Auto price comparison
- Mobile & web version
